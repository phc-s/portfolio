    Especivia

    Especivia é um protótipo de um projeto de aplicativo para dispositivos móveis que se encontra em desenvolvimento, cujo objetivo é criar uma plataforma inovadora que permite clientes com algum tipo de deficiência e motorista particulares possam navegar em um sistema, seguro, confiável e acessível. projeto acessível no GitHub.

Atualmente este projeto se encontra em um estado inicial e planejativo, projeto acessível no GitHub.
